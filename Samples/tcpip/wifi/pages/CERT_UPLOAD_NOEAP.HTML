802.11 Wireless Interface Configuration

This page allows you to update general 802.11 interface settings, and security credentials, for the remote board. To use this form:

  • Select an SSID (network identity) from the list of available SSIDs and hit the Query button. You can then edit the settings as desired then hit the Update button - or -
  • Enter a new SSID, along with all required settings, and hit the Create button.
  • After selecting a suitable SSID, hit the Restart button to bring the 802.11 interface up with the new settings. Note that this action will cause the browser to "hang", since the network connection will be temporarily lost. In this case you can attempt to reload the page, but you may need to change the URL if the IP address was changed

Security Information Section

This section allows you to update the security information for the 802.11 network connection. The information you send via this form will be sent encrypted and is thus secure from eavesdropping and tampering, even if sent over a (currently) unsecure connection. Since secure web pages can be spoofed, always check that your browser is in secure mode - usually there is a "closed padlock" icon or something similar.
